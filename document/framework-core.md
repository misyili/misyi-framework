# framework-core 介绍文档

## 高性能对象拷贝工具
> BeanCopierUtils 

支持对象和 list 拷贝, 不支持基本数据类型和包装类型的拷贝

## 日期工具
> DateUtils

继承 org.apache.commons.lang3.time.DateUtils, 并新增一些自用方法

## 字符串工具
> StringUtils

继承 org.apache.commons.lang3.StringUtils, 并新增一些自用方法
